#### Опыт

С 2011 года занимаюсь веб-разработкой. Писал модули для существующей в веб-студии на тот момент собственной системы управления контентом, занимался вёрсткой.

С середины 2012 года по конец 2014 работал над созданием и поддержкой платформы [InfaCMS](http://infacms.com) — проектирование архитектуры и разработка всех серверных модулей и сервисов. Из-за нехватки опыта некоторые задачи давались с трудом, но основные вещи работают должным образом. На протяжении всей работы оставался обширный круг интересных задач и решений.

> InfaCMS — это сервис, который предоставляет пользователям создавать и управлять своими сайтами. Главная идея состояла в том, чтобы максимально упростить создание простых сайтов со сложной структурой контента на странице. Весь сервис находится в облаке Amazon AWS (используются EC2, S3, RDS). Сама система управления контентом написана на PHP, используя Zend Framework 2.
> 
> На январь 2015 года сервис прекрасно себя чувствует с нагрузкой 100000 - 120000 DAU / Instance (EC2.medium).
